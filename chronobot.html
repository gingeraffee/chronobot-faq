Free vs Paid, in plain language
What ChronoBot does Overview

ChronoBot keeps a single pinned countdown panel in your chosen channel and updates it automatically. It also sends reminders so your server doesn’t have to rely on memory, screenshots, or “wait what day is that again?”

The core countdown system is free. Pro unlocks unlimited events and removes the need to vote for Supporter Perks.

Pricing Pro

ChronoBot Pro (Server): $2.99/month

Pro is a server subscription (Guild) that unlocks premium features for everyone in that server.

Free vs Pro (what you get) Free vs Pro

Free (Included):

  • Pinned countdown board
  • Up to 5 events per server
  • Standard milestone reminders
  • Core commands + setup help (/healthcheck)

Pro (Server Subscription):

  • Unlimited events per server
  • Supporter Perks included (no voting required)
  • Themes, banners, templates, advanced milestones, weekly digest
  • Priority support lane (Pro servers)

Important: If your server has Pro active, voting is optional. You’re already unlocked.

Supporter Perks (vote-locked via Top.gg) Supporter

Supporter Perks unlock after you vote for ChronoBot on Top.gg. These are optional upgrades that improve presentation and automation.

  • Themes (example: /theme)
  • Advanced milestones (example: /milestones advanced)
  • Templates (example: /template save + /template load)
  • Banners (example: /banner set)
  • Weekly digest (example: /digest enable)

If your server has Pro active, voting is not required to access these perks.

Why does Pro exist? Info

Pro helps fund hosting, ongoing development, and support. It also keeps the free tier generous while giving power servers a simple way to scale up.
