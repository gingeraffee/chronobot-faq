What ChronoBot does

Overview

ChronoBot keeps a single pinned countdown panel in your chosen channel and updates it automatically. It also sends reminders so your server doesn't have to rely on memory, screenshots, or "wait what day is that again?"

The core countdown system is free. Pro unlocks unlimited events and removes the need to vote for Supporter Perks.

Free vs Pro (what you get)

Comparison

Free (Included):

  • Pinned countdown board
  • Up to 5 events per server
  • Standard milestone reminders
  • Core commands + setup help (/healthcheck)

Pro (Server Subscription):

  • Unlimited events per server
  • Supporter Perks included (no voting required)
  • Themes, banners, templates, advanced milestones, weekly digest
  • Priority support lane (Pro servers)

Important: If your server has Pro active, voting is optional. You're already unlocked.

Why does Pro exist?

Info

Pro helps fund hosting, ongoing development, and support. It also keeps the free tier generous while giving power servers a simple way to scale up. No ads, no data selling—just quality service.

What are Supporter Perks?

Supporter

Supporter Perks unlock after you vote for ChronoBot on Top.gg. These are optional upgrades that improve presentation and automation.

Pro note: If your server has ChronoBot Pro active, voting is not required. All perks are included.

→

Which features are included in Supporter Perks?

Supporter
  • Themes: /theme — Customize the look of your pinned countdown
  • Advanced milestones: /milestones advanced — Fine-tune reminder timing
  • Templates: /template save + /template load — Reuse event setups
  • Banners: /banner set — Add hero images to your countdown
  • Weekly digest: /digest enable — Get a weekly summary of events

These are quality-of-life upgrades: less repetition, more customization, and nicer presentation.

→

How do I unlock Supporter Perks?

Supporter

1) Run /vote in your server.
2) Click the Vote button (Top.gg will open).
3) Return and run /vote again to confirm you're unlocked.

If Top.gg needs a moment to update, wait a minute and retry. Voting is optional if your server has Pro.
