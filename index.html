What is ChronoBot? Getting Started

ChronoBot helps servers track events with countdowns and reminders. It’s designed to keep one “source of truth” for event timing, post clear countdown updates, and reduce the “when is it again?” spam that haunts every Discord eventually.

What do I need to set up first? Getting Started

Start by choosing where countdowns should live, then configure the countdown channel (example command: /seteventchannel). After that, add a test event and confirm the bot can post and update messages successfully.

If the bot can’t send or pin messages, setup will fail silently in most servers — that’s a permissions issue, not a “mystery bug.”

What are “Supporter Perks” (vote-locked features)? Supporter

ChronoBot is free to use, and the core countdown system is available to everyone. “Supporter Perks” are the optional premium-quality tools (themes, templates, banners, and more) that unlock after you vote for ChronoBot on Top.gg.

Voting is free — it’s just a quick way to support the bot and help more servers find it. Run /vote for the vote button and to confirm your supporter status.

How do I unlock vote-locked features? Supporter

1) Run /vote in your server.
2) Tap the Vote button (Top.gg will open).
3) Come back and run /vote again to confirm you’re unlocked.

If Top.gg needs a moment to update, wait a minute and retry.

Which features are vote-locked (Supporter Perks)? Supporter

Voting unlocks premium tools like:

  • Themes: /theme (changes the look/feel of the pinned countdown embed)
  • Advanced server milestones: /milestones advanced (set your default reminder schedule once)
  • Event templates: /template save + /template load (reuse your perfect setup)
  • Event banners: /banner set (adds a hero image to the pinned embed)
  • Weekly digest: /digest enable (automatic weekly “what’s coming up” post)

These are designed to feel like “server quality-of-life upgrades”: less repetition, more customization, and nicer presentation.

What do themes do? Supporter

Themes change the vibe of the pinned countdown embed — title styling, wording, color accents, and overall presentation.

Examples:

  • /theme theme: neon — bold, energetic presentation
  • /theme theme: minimal — clean, quiet, simple
  • /theme theme: dramatic — theatrical “time is inevitable” energy
  • /theme theme: default — classic Chrono purple
What does “Milestones Advanced” do? Supporter

/milestones advanced sets your server’s default milestone reminder schedule (example: 100, 60, 30, 14, 7, 2, 1, 0). You can also apply that new list to all existing events so everything stays consistent.

Example: /milestones advanced milestones: 100, 60, 30, 14, 7, 2, 1, 0 apply_to_all: true

What are templates, and why would I use them? Supporter

Templates let you save an event’s “setup style” (milestones, repeating reminders, silenced state) and reuse it instantly for new events. Perfect for weekly game nights, monthly meetings, recurring community events, and anything you don’t want to rebuild from scratch.

Examples:

  • /template save index: 1 name: Weekly Meetup
  • /template load name: Weekly Meetup date: 01/15/2026 time: 19:00 event_name: Weekly Meetup
How do banners work (and what kind of URL do I need)? Supporter

Banners add a hero image to the pinned countdown embed. ChronoBot will display the banner for the next upcoming event that has a banner set.

Use a direct image URL (typically ends in .png, .jpg, .jpeg, or .webp).

Example: /banner set index: 1 url: https://example.com/mybanner.png

If your banner isn’t showing, common causes are:

  • The link is not a direct image URL
  • The event with the banner is not the next upcoming event
  • The event is in the past (past events don’t display as “upcoming”)
What is the weekly digest? Supporter

The weekly digest posts a roundup of events happening in the next 7 days. Great for active servers where people miss individual reminders.

Enable it with: /digest enable

If it didn’t post, check that the bot can send messages in the configured channel and that digest is enabled.

I voted, but features still say “locked.” What do I do? Troubleshooting

Quick checklist:

  • Run /vote again (it confirms supporter access).
  • Wait a minute and retry — vote checks may be briefly cached to reduce spam.
  • Make sure you voted on Top.gg while logged into the same Discord account you’re using in the server.

If it still won’t unlock, post in the support server with your Discord username and a screenshot of the issue.

Which permissions does ChronoBot need? Permissions

At minimum, in the channel where countdowns/reminders are posted, ChronoBot typically needs:

  • View Channel
  • Send Messages
  • Embed Links (for rich countdown embeds, if enabled)
  • Read Message History (to find/update its pinned message)
  • Manage Messages or Pin Messages (only if your setup uses a pinned countdown message)

If you can run commands but nothing posts, check channel overrides and role hierarchy first. Discord permission layering is… a whole saga.

“I couldn’t create or access the pinned countdown message.” What does that mean? Troubleshooting

That message almost always means a permissions issue in the countdown channel. ChronoBot can’t create the pinned message, can’t read message history to find it, or can’t pin/update it.

Fix: confirm the bot has the needed channel permissions (especially Send Messages, Read Message History, and Manage Messages/Pin Messages if pinning is required).

How do events work in ChronoBot? Events

ChronoBot stores events (name + date/time + optional settings) and can display them as a list with an index. Some management commands reference events by index for speed and simplicity.

Best practice: keep event names specific (“Guild Raid Night”, “Giveaway Ends”, “Release Day”) so reminders stay readable.

What happens when an event time passes? Events

Passed events are automatically cleaned up after they start (with a short grace window), so the pinned countdown stays fresh. If your server prefers manual cleanup, you can also use /archivepast.

How do reminders work? Reminders

Reminders are scheduled notifications for upcoming events. Some reminders fire at milestone intervals (for example: days before), and some can target a channel, role, or everyone depending on your server configuration and Discord permissions.

If reminders don’t ping who you expect, confirm the bot has permission to mention roles (and that role mentions are allowed in that channel).

Why don’t I see autocomplete options for event index fields? Troubleshooting

Discord’s autocomplete only appears when a command option is explicitly wired to an autocomplete handler, and some option types behave differently depending on how the command is defined.

If you expected autocomplete (example: event index suggestions) and it’s not showing, it may be a command-definition issue—or a Discord client quirk. Report it in the support server with the command name, screenshots, and whether you’re on desktop/mobile.

Which timezone does ChronoBot use? Time Zones

ChronoBot uses a configured default timezone for interpreting event times. If your server spans multiple regions, it’s worth standardizing (e.g., “All events are in Central Time”).

Daylight Saving Time can cause confusion twice a year. If something looks off by exactly one hour, DST is the prime suspect.

What data does ChronoBot store? Privacy

ChronoBot typically stores server configuration needed to function (like which channel to post in) and event/reminder data you create. It does not need (or want) your private messages.

If you need a formal privacy statement for Top.gg, ask in the support server and we’ll provide the current policy text.

How do I request a feature or report a bug? Support

Use the support server and post in the appropriate channel (support / bugs / suggestions). For bugs, include reproduction steps and screenshots/logs if possible.

Support inquiries receive a response within 24 hours or less.
