What is ChronoBot?

Getting Started

ChronoBot (aka Chromie) helps servers track events with countdowns and reminders. Each countdown channel keeps one pinned "source of truth" for upcoming dates, plus automated reminders so your server doesn't get haunted by "wait when is it again?" every five minutes.

Everything is per channel: each countdown channel has its own events, theme, timezone, and settings.

→

What do I need to set up first?

Getting Started

Three-step setup (about 30 seconds):

  • In the channel you want the countdown → /seteventchannel
  • Add an event → /addevent date: 04/12/2026 time: 09:00 name: Game Night 🎲
  • Chromie pins a countdown and keeps it updated automatically

Dates use MM/DD/YYYY and times use 24-hour HH:MM. Each channel has its own timezone (default America/Chicago) — change it in /countdown → Timezone.

If the pinned message doesn't appear, it's almost always a channel permission issue. Run /healthcheck, or check View Channel, Send Messages, Embed Links, Read Message History, and Manage Messages/Pin Messages.

What are the tiers (Free, Supporter, Pro)?

Pricing

Chromie has three tiers. The core countdown system is always free.

🆓 Free — up to 3 events, 1 countdown channel, all core features, milestone reminders, and full event management.

⭐ Supporter — up to 5 events, premium themes, and custom event banners. Unlock free by voting on Top.gg (lasts 12 hours per vote).

💎 Pro ($2.99/month) — unlimited events, a separate countdown in every channel, recurring reminders, event templates, event duplication, weekly digest, build-your-own themes, and all Supporter features permanently (no voting). Subscribe via Discord's Server Subscription.

See the Features & Pricing page for the full breakdown.

→

What are "Supporter Perks" and how do I unlock them?

Supporter

Supporter Perks are optional upgrades that unlock for free when you vote for Chromie on Top.gg. Each vote unlocks Supporter for 12 hours — vote again afterward to keep access.

To unlock:

1) Run /vote to get the link and check your status.
2) Click the link and vote (takes about 10 seconds).
3) Return and run /vote again to confirm you're unlocked.

If Top.gg needs a moment to update, wait a minute and retry. Admins can use /vote_debug to troubleshoot.

How do I get Pro, and does voting still matter once I have it?

Pro

Chromie Pro is $2.99/month per server, purchased through Discord's Server Subscription (in the bot's Discord Store). Pro unlocks for the entire server — all admins get Pro features — and you can cancel anytime in Discord's subscription settings.

Check status with /pro_status. If Discord and Chromie ever look out of sync after subscribing, an admin can run /sync_subscription.

Once Pro is active, voting is optional. All Supporter perks are included permanently — no 12-hour vote window to maintain. (Voting is still appreciated and helps Chromie grow.)

I changed a setting, but the pinned embed didn't update. Why?

Troubleshooting

The pinned countdown refreshes automatically, but if a theme or setting change doesn't show right away, run /update_countdown to force a refresh, or wait for the next auto-refresh cycle.

If it still doesn't change, run /healthcheck and confirm Chromie can Read Message History and Embed Links in that channel.

→

How do banners work (and what kind of URL do I need)?

Customization

Banners add a hero image to the pinned countdown embed. Set one per event in /event → Banner. Chromie displays the banner for the next upcoming event that has a banner set.

Use a direct image URL ending in .png, .jpg, .jpeg, or .webp.

If your banner isn't showing:

  • The link isn't a direct image file
  • The banner is on an event that isn't the next upcoming event
  • The event is in the past
  • The bot lacks Embed Links permission

Banners are a Supporter/Pro perk.

Which permissions does ChronoBot need?

Permissions

In the countdown channel, ChronoBot typically needs:

  • View Channel
  • Send Messages
  • Embed Links
  • Read Message History (to find/update its pinned message)
  • Pin Messages or Manage Messages (if pinning is used)
  • Manage Messages (to auto-delete milestone announcements and repeating reminders after the event)

If commands work but nothing posts, check channel permission overrides first. Discord's permission layering is… dramatic.

→

"I couldn't create or access the pinned countdown message." What does that mean?

Troubleshooting

That message almost always means a permissions issue in the countdown channel. ChronoBot can't create the pinned message, can't read history to find it, or can't pin/update it.

Fix: confirm Send Messages, Read Message History, Embed Links, and Pin Messages/Manage Messages.

→

Why can't some members see or use /event, /countdown, or /seteventchannel?

Permissions

The setup and management commands (/seteventchannel, /event, /countdown, /purgeevents, etc.) require the Manage Server permission and are hidden from members who don't have it. That's intentional, so regular members can't reconfigure your countdowns.

Anyone can still use the read-only commands like /listevents, /nextevent, and /chronohelp. If an admin has Manage Server but the command still won't run, check for a channel-level permission override.

How do events work in ChronoBot?

Events

Chromie stores events (name + date/time + optional settings) per countdown channel and shows them in a numbered list (/listevents). Open /event to pick an event and manage everything about it — edit, delete, milestones, reminder time, silence, owner, owner DMs, banner, and (on Pro) repeat and duplicate.

Best practice: keep names specific ("Guild Raid Night", "Giveaway Ends", "Release Day") so reminders stay readable.

→

What happens when an event passes?

Events

The pinned countdown moves on to the next upcoming event, and past events auto-remove after they pass so the list stays tidy. You can also clear them out yourself anytime with /archivepast (across all channels).

Chromie can also clean up its own reminder posts — see the auto-delete question below.

→

How do reminders and milestones work?

Reminders

Reminders fire automatically at milestone intervals — by default 100, 60, 30, 14, 7, 2, 1, and 0 days before an event — and post in the countdown channel. You can also post one on demand with /remindall.

Per event, in /event you can:

  • Milestones — set custom milestone days or reset to default
  • Reminder time — set the day-of reminder time (or clear it)
  • Silence — pause an event's reminders without deleting it

Pick who gets pinged with /countdown → Mention role. If reminders don't ping who you expect, confirm role mentions are allowed and the bot can mention them in that channel.

→

Do milestone and reminder posts get deleted automatically?

Reminders

By default, yes — to keep the countdown channel from getting cluttered, Chromie auto-deletes its milestone and reminder announcements 24 hours after they're posted. This is a per-channel setting.

To change it, run /countdown → Auto-delete posts and choose Auto-delete after 24h (default) or Keep forever.

Required permission: Chromie needs Manage Messages in the channel to delete posts. If it can't, the posts simply stay.

→

What are repeating reminders?

Reminders

Repeating reminders post on a recurring schedule for an event, in addition to milestone reminders. Set them up in /event → Repeat.

Recurring reminders are a Chromie Pro feature.

Best practices (recommended server setup)

Suggestions
  • Create a dedicated events channel (ex: #events or #countdowns).
  • Set the channel's timezone right away in /countdown → Timezone.
  • Keep event names specific so reminders make sense.
  • If you use role pings, create one role (ex: "Event Pings"), let people opt in, and set it via /countdown → Mention role.
  • On Pro, use /event → Duplicate (or templates) for recurring formats so you don't rebuild everything.
  • Use a banner only for "headline" events so the pinned embed stays clean.

Can I have more than one countdown?

Events

Your first countdown channel is free. Chromie Pro unlocks a separate countdown in every channel — each with its own events, theme, timezone, and settings. Just run /seteventchannel in each channel you want a countdown in.

Can I add events from DMs?

Events

Yes. Run /linkserver inside the server first to link your DMs to it (requires Manage Server). After that, you can DM Chromie /addevent and it'll add the event to your linked server.

What data does ChronoBot store?

Privacy

ChronoBot stores server configuration needed to function (like which channels to post in) and the event/reminder data you create. It does not read your private messages for content. See the Terms & Privacy page for details.
